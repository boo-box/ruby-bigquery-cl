Class BQ::Table
In: lib/ruby-bigquery-cl/table.rb

This class can be used to create and delete tables.

Methods

Public Class methods

create(table, schema)

Create a new table. Call this class method with the name of the table and one shema array with collumns hashes following this example: shema = [ 

 {:id" => "name", :type => "string", :mode => "REQUIRED"},
 {:id" => "age", :type => "integer"},
 {:id" => "weight", :type => "float"}
 {:id" => "is_magic", :type => "boolean"}

] A hash is returned with the API response.

delete(table)

Delete table. This class method never fails! A hash is returned with the API response.
